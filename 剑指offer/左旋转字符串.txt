//对于一个给定的字符序列S，请你把其循环左移K位后的序列输出。
//例如，字符序列S=”abcXYZdef”,要求输出循环左移3位后的结果，即“XYZdefabc”。是不是很简单？OK，搞定它！
string LeftRotateString(string str, int n) 
{
	if(str.size() == 0)
		return "";
	string s;
    s += str.substr(n,str.size()-n);
	s += str.substr(0,n);
	return s;
}
